# Cystol
Anonymous chat platform made in Java. No tracking.

# Server Hosting

**Requirements**
- Java 17 or higher 🍵
- Working Windows/Mac/Linux computer 💻

**Installing Java**
Linux
```sh
> sudo apt install openjdk-17-jre
```

### Now, running the application

Windows

```powershell
PS C:\Users\sexy > java -jar .\server.jar
```

```sh
> java -jar ./server.jar
```
